```
         🧠 MASTER (oversees all)
                ↓
    EXECUTOR ←→ CREATOR ←→ MEMORY
```

## HOW AN INTERACTION WORKS

Example: Wyatt asks "Should I take the Moat job?"

1. **Master** → Coordinates the response
2. **Memory** → Pulls all past Moat conversations
3. **Analyst** → Checks income impact ($ vs Spark)
4. **Strategist** → Weighs pros/cons
5. **Faith** → Checks values alignment
6. **Core** → Gives warm, balanced answer
7. **Creator** → (if building something)
8. **Executor** → (if running a task)
9. **Coach** → (if motivation needed)
10. **Master** → Logs interaction, backs up
